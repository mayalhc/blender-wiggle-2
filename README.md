# Wiggle 2: RTX Edition (Blender 5.0+)

**Wiggle 2: RTX Edition** is a community-maintained fork of the renowned **Wiggle 2** add-on, originally created by the talented **[shteeve3d]([https://github.com](https://github.com/shteeve3d/blender-wiggle-2))**. 

As the original project is no longer compatible with the latest Blender API, I have taken over the maintenance to ensure this essential tool continues to work perfectly for professional production in **Blender 4.2 LTS, 5.0, and 5.1+**.

---

## ✨ What's New in RTX Edition
While preserving the core logic of the original Wiggle 2, this version introduces:

*   **Modern API Support**: Fully updated for Blender 5.0/5.1 and converted to the new Extension system.
*   **Physics Blending (Sim Mix)**: Mix keyframed animation and live physics seamlessly.
*   **Safety Guard**: Adaptive damping to prevent bone "explosions" during fast movements.
*   **Production Optimized**: Specifically tuned for high-density rigs (like long hair) to eliminate viewport lag.
*   **Multi-Bone Sync**: Edit properties for all selected bones at once for faster workflow.
